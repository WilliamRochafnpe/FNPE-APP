
# PONTO DE RESTAURAÇÃO: Cópia_FNPE
**Data:** 22 de Maio de 2025 (Simulada)
**Versão:** 1.0.0-GOLDEN

## Funcionalidades Seladas neste Ponto:
1. **Dashboard Regional**: Gráficos Recharts processando dados do Norte (AC, AM, AP, PA, RO, RR, TO).
2. **ID Norte Digital**: Sistema de emissão, aprovação condicionada ao financeiro e download de PDF.
3. **Certificação de Eventos**: Fluxo completo de solicitação, análise administrativa e publicação no calendário.
4. **Comunicação Admin**: Disparo de notificações em massa ou individuais com auxílio de IA (Gemini).
5. **Assistente IA**: Integrado com busca no Google e conhecimento das resoluções da FNPE 2026.
6. **Sincronização**: Lógica híbrida preparada para Supabase com fallback automático para LocalStorage.

## Estrutura de Arquivos Protegida:
- `App.tsx`: Gerenciamento de rotas e estado global.
- `db.ts`: Sementes de dados e lógica de persistência.
- `pages/RegionalStats.tsx`: Inteligência de dados geográficos.
- `pages/AdminCommunication.tsx`: Gestão de mensagens com correção de tipos.
- `geminiService.ts`: Configuração do motor de IA.
